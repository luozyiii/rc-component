演示组件所有色彩

## 何时使用

## 属性

| 属性      | 说明             | 类型                          | 默认值 |
| --------- | ---------------- | ----------------------------- | ------ |
| className | 语义化结构 class | -                             | -      |
| style     | 语义化结构 style | -                             | -      |
| data      | 数据             | `{name: 'red', value: red}[]` | -      |
