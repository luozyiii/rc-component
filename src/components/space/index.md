# Space 间距

设置组件之间的间距。

## 何时使用

避免组件紧贴在一起，拉开统一的空间。

## 示例

<code src="./demos/demo1.tsx"></code>

## 属性

| 属性      | 说明                                 | 类型                      | 默认值     |
| --------- | ------------------------------------ | ------------------------- | ---------- |
| direction | 间距方向                             | `horizontal \| vertical ` | horizontal |
| wrap      | 是否自动换行，仅在 horizontal 时有效 | `boolean`                 | true       |
| size      | 间距大小                             | `number`                  | 12         |
