# 基础色

## Dust Red / 薄暮

斗志、奔放

```jsx
import React from 'react';
import { Palette, createColorData } from 'rc-component';
import { red } from '@leslies/css';

const colorKey = 'red';
const colorsData = createColorData(colorKey, red);
export default () => <Palette data={colorsData} />;
```
